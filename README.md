# Smart IT Asset Management Application

A custom ServiceNow application engineered to streamline, track, and automate the lifecycle of high-value corporate hardware assets. This solution bridges the gap between catalog tracking and database integrity by utilizing custom relational schemas and automated flow processing.

## 🚀 Key Features
* **Custom Hardware Asset Schema:** Extends baseline capabilities to provide tracking for specialized model categories, serial numbers, and stockroom assignments.
* **Optimized Form & List Layouts:** Engineered intuitive user views ensuring clean visibility into asset-specific data tracking.
* **Automated Asset Approval Flow:** An end-to-end multi-stage flow built in Flow Designer that handles catalog intake, processes approvals, dynamically updates asset records, and triggers confirmation notifications.

## 🛠️ Technical Implementation
* **Platform:** ServiceNow (Utah/Vancouver/Washington)
* **Tools Used:** Studio, Flow Designer, Form Designer, Service Catalog
* **Core Tables:** Extended IT Asset tracking structures (`x_1958828_smart_0_hardware_asset`)
